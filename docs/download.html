Baixar Circuito Ferradura (Windows)

Release atual: v2.0.0. Você será redirecionado para a página de Releases. Baixe o arquivo CircuitoFerradura.exe e, se desejar, o checksum.sha256 para verificar a integridade.

Na página do Release você encontra:

  • CircuitoFerradura.exe — aplicativo Windows (executar em duplo clique)
  • checksum.sha256 — para validar o arquivo (recomendado)
  • checksum.md5 — alternativa de verificação
  • INSTALACAO.md — guia rápido de instalação e uso

Para verificar o SHA-256 no PowerShell: Get-FileHash -Path .\CircuitoFerradura.exe -Algorithm SHA256. Compare o resultado com o valor em checksum.sha256.

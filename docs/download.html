Baixar Circuito Python (Windows)

Você será redirecionado para a página de Releases. Baixe o arquivo CircuitoPython.exe e, se desejar, o checksum.sha256 para verificar a integridade.

Na página do Release você encontra:

  • CircuitoPython.exe — aplicativo Windows (executar em duplo clique)
  • checksum.sha256 — para validar o arquivo (recomendado)
  • checksum.md5 — alternativa de verificação
  • INSTALACAO.md — guia rápido de instalação e uso

Para verificar o SHA-256 no PowerShell: Get-FileHash -Path .\CircuitoPython.exe -Algorithm SHA256. Compare o resultado com o valor em checksum.sha256.

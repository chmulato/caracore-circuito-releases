  • Modules e packages
  • Tratamento de exceções

Orientação a Objetos

  • Design patterns

Desenvolvimento Web

24 horas • 3 semanas
Tópicos principais:
  • Flask framework
  • Templates e Jinja2
  • Formulários e validação
  • SQLAlchemy ORM
  • Authentication & sessions

APIs e Banco de Dados

22 horas • 3 semanas

  • PostgreSQL e MongoDB
  • Migrations e seeding
  • API testing

Deploy e Produção

20 horas • 2 semanas

  • Security & performance
